# Voice Truck Assistant 🚚

## Features
- Voice-based interaction
- Driver-safe (no screen needed)
- Commands:
  - Next stop
  - Traffic update
  - Send delay message

## Tech Used
- Python
- SpeechRecognition
- ElevenLabs (text-to-speech)

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Add your ElevenLabs API key in voice_output.py

3. Run:
   python main.py

## Notes
- Uses simulated data for demo
- Designed for simplicity and reliability
